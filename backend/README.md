## Po pobraniu projektu 
1. w katalogu ``` ./backend``` stwórz wirtualne środowisko 
```python -m venv venv```
2. Aktywuj wirtualne środowisko 
dla Windowsa ```venv\Scripts\activate```
3. Zainstaluj paczki 
```pip install django```
4. Uruchomienie servera
  ```  python manage.py runserver  ```
5. ...
